### 프로젝트의 목적

- 협업 프로젝트는 Git을 통해 서로의 바이오 데이터를 공유하고 이를 분석하여 유의미한 데이터를 생성

- 축적된 데이터를 토대로 새로운 관점으로 데이터 해석

- 해석한 데이터를 토대로 맞춤 제약 서비스 및 질병 조기 발견 분야 등에 적용 가능성 분석
